# Project Documentation

This directory contains the **final consolidated report** for the capstone project  
**“ERP-Focused Analysis of APT41 Supply-Chain Intrusion Campaigns”**, completed as part of the  
**ICDFA Cyberwarfare Program (ACW905)**.

All research findings, analysis, visual mappings and supporting references for this project are documented within a single comprehensive report.

---

## Contents

### Final Capstone Report

This report serves as the **primary and complete deliverable** for the project. It includes:

- Executive summary and project briefing  
- Project objectives and problem statement  
- Detailed adversary profile of APT41  
- ERP-focused intrusion lifecycle analysis  
- MITRE ATT&CK technique mapping tables and visual matrix  
- ERP monitoring and mitigation priority brief  
- Conceptual ERP threat telemetry and deception framework   
- Full references to publicly available threat intelligence sources  

---

## Scope and Methodology

The analysis presented in this report is based exclusively on **publicly available threat intelligence**, vendor publications and academic sources.  
The project adopts a defensive, intelligence-driven approach aligned with enterprise application security and cyber threat intelligence best practices.
